Predicción con Datos Temporales (36519). Grado en Inteligencia y Analítica de Negocios (GBIA)


Hola a tod@s y bienvenid@s a la página web del curso.


Esta es la página web de la edición 2020-21 del curso de Predicción con Datos Temporales (GBIA) de la Universitat de València. El repositorio para crear esta web lo tenéis aquí en Github.

Resumen del curso: la idea principal es aprender a manejarse con datos con estructura temporal (en contraposición a datos de corte transversal (que has visto en el primer semestre con Paco Goerlich) o espacial. Veremos cómo describir una serie temporal, como ajustarla a un modelo, cómo hacer predicciones y, lo que es más importante, a valorar la calidad de las predicciones desde un enfoque de Machine Learning.

Para más información de este módulo, por favor visita la sección Guía del curso de esta página web o descárgate la Guía Docente desde la web de la Universitat de València.
